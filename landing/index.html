Why DAOL Concierge

차별화 포인트

01

Claude Tool Use 자율 에이전트

한 메시지에서 1~N 개 호텔 액션을 Claude 가 자율 호출. "수건 두 개·모닝콜 7시·체크아웃 1시간 연장" 한 줄 → 3 부서 동시 라우팅. 8 tools (어메니티/HK/레이트/분실물/VOC/대여/주차/대화).

02

RAG 호텔 챗봇 + 출처 인용

호텔 매뉴얼·시설·운영 KB 14 항목 임베딩 → 유사도 검색 → Claude 답변 + [출처: §section] 마커. CJK-safe BM25-lite, 한·영·일·중 동일 검색 경로.

03

다국어 양방향 통역 (4×4 = 12 페어)

스태프 전 페이지에 통역 위젯 상주. 외국어 게스트 메시지를 한국어로 즉시 전환, 한국어 응대를 게스트 언어로 정중한 호텔 톤 유지하며 변환.

04

매니저 일일 AI 리포트

출근 즉시 어제 운영 데이터(태스크·SLA·VOC·분실물·당직)를 Claude 가 1페이지 마크다운 분석. 📊 KPI 요약 / ⚠️ 이상 패턴 / 🎯 권장 액션 3 섹션.

05

멀티 테넌시 + PMS 어댑터

PROP_CD × CMPX_CD 스코프 단일 배포. PmsRemoteApi(REST)/PmsSyncAdapter 어댑터로 사내 PMS·타 PMS·standalone 3 모드 모두 지원.

06

실시간 SLA + 감사 로그

초과 시 WebSocket 에스컬레이션 푸시 · 모든 도메인 변경을 before/after JSON 으로 감사 기록.

객실 QR 또는 태블릿에서 어메니티·하우스키핑·레이트 체크아웃을 셀프 처리. Claude Tool Use 에이전트가 한 메시지에서 여러 호텔 액션을 자율 실행하고, RAG 챗봇이 호텔 지식베이스를 인용하여 답변합니다. 한·영·일·중 다국어 통역 위젯과 매니저 일일 AI 리포트로 운영 효율을 극대화합니다. LLM 키 없이도 룰 기반 폴백으로 오프라인 동작합니다.
